arXiv data (technical research)
===============================

Data collection and processing pipeline for arXiv data,
principally for the `arXlive <https://arxlive.org>`_ platform.
This pipeline orchestrates the collection of arXiv data, enrichment
(via MAG and GRID), topic modelling, and novelty (lolvelty) measurement.
